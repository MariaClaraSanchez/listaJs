Exercício 7

Exercício 14

Explique quais as vantagens e desvantagens das linguagens fracamente tipadas e fortemente tipadas levando em consideração as linguagens de programação JavaScript e Java.

Fortemente tipada

Uma linguagem fortemente tipada requer que, na declaração de uma variável ou de uma função, seja fornecido o tipo de retorno, ou seja, uma variável não será convertida automaticamente de um tipo para outro. E a linguagem Javascript não é fortemente tipada, pois podemos declarar ela sem um tipo, por exemplo: var nome = Jádyla;
Porém a linguagem Java, é considerada fortemente tipada, pois é obrigatóiro a informação do tipo do reterno. Por exemplo: String nome = Maria;

Vantagens

  • Funções com tipos de dados bem definidos;
  • Alocação de espaço necessário de memória para a variável;
  • Possuem maior Aprendibilidade por possuir códigos mais legíveis;
  • Pura versatilidade.

Desvantagens

  • Limitação no tamanho das variáveis;
  • Difícil de aprender.

Fracamente tipada

Uma linguagem fracamente tipada, significa que o compilador descobre o tipo de cada variável em tempo de compilação, ou seja, apenas descobrem os tipos de variáveis em tempo de execução.

Vantagens

  • A maneira mais rápida e fácil de codificar;
  • Programação moderna;
  • Flexibilidade de alterar o conteúdo da variável quando bem desejar.

Desvantagens

  • Problema na hora de dar manutenção;
  • Problema com os tipos de variáveis.
  • Conversões de Tipos podem gerar erros.
